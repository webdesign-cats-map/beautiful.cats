# 接口文档

**系统名称**：校园流浪猫 WebGIS 系统  
**接口版本**：v1.0  
**基础地址**：`http://localhost:5000/api`  
**数据格式**：JSON  

---

## 通用响应格式

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 200=成功，400=失败，404=不存在 |
| msg | string | 提示信息 |
| data | any | 返回数据 |

---

## 系统接口

### 获取配置

```
GET /api/config
```

**响应示例**
```json
{
  "code": 200,
  "data": {
    "amap_key": "your_key",
    "version": "1.0.0",
    "system_name": "校园流浪猫WebGIS系统"
  }
}
```

## 用户认证

### 注册

```
POST /api/auth/register
```

**请求体**
```json
{
  "username": "testuser",
  "password": "123456",
  "nickname": "测试用户",
  "email": "test@example.com"
}
```

**响应**
```json
{
  "code": 200,
  "msg": "注册成功",
  "data": { "username": "testuser" }
}
```

### 登录

```
POST /api/auth/login
```

**请求体**
```json
{
  "username": "testuser",
  "password": "123456"
}
```

### 登出

```
POST /api/auth/logout
```

## 猫咪档案

### 获取猫咪列表

```
GET /api/cats?page=1&limit=20&status=healthy&gender=male
```

**Query 参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| page | int | 页码，默认1 |
| limit | int | 每页数量，默认20 |
| status | string | 健康状态：healthy/injured |
| gender | string | 性别：male/female |

**响应**
```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": "cat_001",
        "name": "橘橘",
        "gender": "male",
        "color": "橙色虎斑",
        "personality_tags": ["亲人", "爱吃"],
        "health_status": "healthy",
        "neutered": true,
        "location": {
          "lng": 116.397428,
          "lat": 39.909728,
          "address": "图书馆门口"
        },
        "description": "非常亲人的大橘猫",
        "checkin_count": 12,
        "last_seen": "2024-05-01 14:30:00",
        "created_at": "2024-01-01 10:00:00"
      }
    ],
    "total": 4
  }
}
```

### 获取猫咪详情

```
GET /api/cats/{cat_id}
```

### 新增猫咪（管理员）

```
POST /api/cats
```

**请求体**
```json
{
  "name": "新猫",
  "gender": "female",
  "color": "纯白",
  "personality_tags": ["亲人"],
  "health_status": "healthy",
  "neutered": false,
  "location": {
    "lng": 116.397,
    "lat": 39.909,
    "address": "教学楼A区"
  },
  "description": "猫咪描述"
}
```

### 更新猫咪（管理员）

```
PUT /api/cats/{cat_id}
```

请求体同新增，字段可选。

### 删除猫咪（管理员）

```
DELETE /api/cats/{cat_id}
```

## 打卡记录

### 获取打卡列表

```
GET /api/checkins?cat_id=cat_001&user_id=xxx&type=encounter&page=1
```

| 参数 | 说明 |
|------|------|
| cat_id | 按猫咪筛选 |
| user_id | 按用户筛选 |
| type | encounter/feed |

### 新增打卡

```
POST /api/checkins
```

**请求体**
```json
{
  "cat_id": "cat_001",
  "type": "encounter",
  "location": {
    "lng": 116.397,
    "lat": 39.909,
    "address": "图书馆门口"
  },
  "photo": "base64或url",
  "note": "今天橘橘心情很好",
  "user_id": "user_001"
}
```

**响应**
```json
{
  "code": 200,
  "msg": "打卡成功，等待审核",
  "data": {
    "id": "checkin_xxx",
    "status": "pending",
    "created_at": "2024-05-01 15:00:00"
  }
}
```

### 审核打卡（管理员）

```
PUT /api/checkins/{checkin_id}/approve
```

**请求体**
```json
{ "status": "approved" }
```

status 取值：`approved`（通过）/ `rejected`（拒绝）

  "code": 200,

    "total_cats": 4,
    "total_checkins": 23,
    "pending_reviews": 2,

    { "lng": 116.397, "lat": 39.909, "weight": 5 },

## 错误码说明

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录/Token无效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
